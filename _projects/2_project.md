CT-PIFO (Congruity Tree-based PIFO) explores packet scheduling architectures for programmable high-speed network devices. The work is motivated by the need to improve flow completion time while keeping scheduling logic practical for hardware implementation.

The project investigates:

- PIFO-inspired queueing and scheduling abstractions;
- hardware-conscious data structures for packet ordering;
- flow completion time improvements under high-speed traffic;
- resource and latency trade-offs in data-plane schedulers.

This line of work connects algorithmic scheduling ideas with the constraints of real hardware pipelines.
