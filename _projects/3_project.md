This project area covers the architectural design and evaluation of programmable packet-processing pipelines for high-speed networks. The emphasis is on FPGA-based implementations where performance, timing closure, memory layout, and resource utilization all matter.

Topics include:

- hardware/software co-design for network systems;
- packet parsing, classification, and scheduling pipelines;
- performance measurement of throughput and latency;
- practical FPGA constraints in deployable networking systems.

The long-term objective is to build network functions that are both fast and realistic to deploy on programmable hardware.
