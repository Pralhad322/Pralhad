This project focuses on designing intrusion detection system (IDS) components that can operate at high throughput on FPGA platforms. The goal is to move expensive packet-processing and inspection tasks closer to the data plane while preserving practical resource usage.

Key research directions include:

- line-rate packet parsing and feature extraction;
- hardware-aware IDS architecture design;
- throughput, latency, and FPGA resource trade-off analysis;
- deployable network-security pipelines for high-speed traffic.

The broader motivation is to make network security mechanisms fast enough for modern links without relying only on software-based inspection paths.
